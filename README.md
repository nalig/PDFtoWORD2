# 🏠 Attics — Share More, Own Less

Modern landing page for the **Attics** app — a peer-to-peer rental marketplace for everyday items.

## 🌐 Live

Hosted via [GitHub Pages](https://pages.github.com/).

- 🇬🇧 English: `/index.html`
- 🇳🇱 Nederlands: `/nl/index.html`

## 📁 Structure

```
├── index.html              # English landing page
├── nl/
│   └── index.html          # Dutch landing page
├── assets/
│   ├── css/
│   │   └── style.css       # Shared styles
│   ├── js/
│   │   └── script.js       # Shared interactivity
│   └── img/
│       └── logo.png        # Brand logo
├── .gitignore
└── README.md
```

Enable GitHub Pages in repo settings → **Source: Deploy from a branch** → select `master` / `main`, root `/`.
